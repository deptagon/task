# Multi-Function Python Assignment  

A collection of small Python programs demonstrating basic control flow, user input handling and logic.  
This project includes a variety of tasks (e.g., comparing numbers, calculating bills, games, etc.) — helpful for practicing fundamental programming concepts.

## Programs Included  

| # | Program / Task | Description |
|---|----------------|-------------|  
| 1 | Greatest of Three Numbers | Prompts user for three numbers and prints which one is greatest |  
| 2 | Month Name from Number | Converts a month number (1–12) into the month name |  
| 3 | Swap Two Variables | Swaps values of two inputs and displays before & after |  
| 4 | Movie Ticket Price Calculator | Computes ticket price (or free) based on user’s age and membership status |  
| 5 | Electricity Bill Calculator | Calculates electricity bill according to different unit-based tariff slabs |  
| 6 | Even/Odd and Positive Check | Checks if an entered number is positive and determines whether it is even or odd |  
| 7 | Compare Two Numbers & Sign Check | Compares two numbers; if equal — determines if positive, negative or zero |  
| 8 | FizzBuzz | Prints “Fizz”, “Buzz”, “Fizz Buzz”, or the number according to divisibility by 3 and/or 5 |  
| 9 | Discount Calculator | Applies discount on total purchase amount based on membership and total value |  
| ... | (Other small tasks) | Additional small programs showcasing conditional logic and user input |  

> **Note:** All tasks are consolidated so you can run one at a time, depending on selection — avoiding script-overload.
